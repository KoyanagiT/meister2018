# meister2018

meister2018で作った作品

怪しそうな文言や，マルウェアを送信してくるなりすましアカウントを検知する，Line，Twitter Bot，
「なりすまし検知マン」．言葉が送られた場合はこちらで用意したデータセットを参照し，危険か判定する．ファイルの場合はハッシュ値を求めてそれをvirustotalと呼ばれるウイルスを検知できるサイトに送る．

現在サーバーは止まっているため，返答しない．

担当は送られてきたファイルを変換してvirustotalに送信し，ウイルスかどうか判定するプログラムなど．LineBOT製作には関わっていない．
